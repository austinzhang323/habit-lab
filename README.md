# Habit Lab

A fullstack Next.js app that tracks daily habits and correlates them with performance metrics like energy, mood, and focus.

## Features
- Daily check-ins (sleep, energy, mood)
- Habit tracking with streaks
- Performance dashboard with trends
- End-to-end testing with Playwright

## Tech Stack
- Next.js (App Router)
- TypeScript
- Prisma + PostgreSQL
- Tailwind CSS
- Playwright

## Why I Built This
I wanted to explore how daily habits impact performance and build a system that surfaces meaningful insights from personal data.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or

```
