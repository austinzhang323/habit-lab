# Habit Lab

A fullstack Next.js app for tracking daily habits with a rolling completion grid and a stats dashboard.

## Features
- Habit tracker with a rolling 30-day completion grid
- Dashboard with completion rates, streaks, and recent activity
- Habit management with categories
- End-to-end testing with Playwright

## Tech Stack
- Next.js (App Router)
- TypeScript
- Prisma + PostgreSQL
- Tailwind CSS
- Playwright

## Why I Built This
I wanted to explore how daily habits impact performance and build a system that surfaces meaningful insights from personal data.

## Getting Started
